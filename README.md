# 🛡️ Drishti-AX: Visual Intelligence Pulse
> **The definitive Accessibility Forensic & Remediation Command Center.**

Drishti-AX is a high-performance analytics dashboard designed to transform raw accessibility JSON audits into actionable engineering roadmaps. It doesn't just report failures; it simulates the path to digital redemption for large-scale web ecosystems.

## 🚀 Key Features

* **⚡ Intelligence KPIs:** Instant visibility into Average Health, Total Technical Debt, JS Stability, and Content Complexity.
* **💀 Wall of Shame:** Automatically ranks the bottom 10 worst-performing URLs based on weighted violation counts.
* **🔬 Forensic Remediation Hub:** A deep-dive engine that extracts broken code snippets directly from source files and provides instant "Suggested Patches."
* **📈 Redemption Simulator:** A predictive tool that allows stakeholders to select systemic fixes (e.g., `button-name`) and see a projected jump in the state’s global health score.
* **🕸️ WCAG Failure Matrix:** A heatmap identifying "Systemic Killers"—issues present across every site in the sector due to faulty templates.
* **🌑 Midnight Classic Theme:** A high-contrast, professional dark mode designed for modern Security/Operations Centers.
